Servant - easy APIs.

Servant is a lightweight tool that makes building Haskell API services a breeze. With only the thinnest sliver of boilerplate, you get a type-safe service that is capable of generating client-side code and documentation for free.

Your API

When you use Servant, your API is represented as a type:

type MyAPI = "date" :> Get Date
        :<|> "time" :> Capture "tz" Timezone :> Get Time

This represents an API with two endpoints, both of which accept only GET requests, and which return a (JSON representation of) a Date and Time object, respectively.
